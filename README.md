Intent-Aware & Style-Adaptive AI (VinnovateIT Round 2)

Hi! This is my project for the recruitment round. It detects the intent of your question (using a Zero-Shot model) and answers in a specific persona (using Gemini).

Hugging Face (Intent Detection)

Google Gemini (Response Generation)

 How to Run (For Evaluators)

I removed my API key for security. Please add yours to test it!

Install: pip install -r requirements.txt (or py -m pip ... on Windows)

Config: Open app.py (Line 15) and paste your Gemini API Key.

Run: streamlit run app.py
